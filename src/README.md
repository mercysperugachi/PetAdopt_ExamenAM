# PetAdopt - Arquitectura Clean

Este proyecto sigue la arquitectura Clean y el stack tecnológico definido en SKILL.md.

## Estructura de carpetas

- src/domain: Reglas de negocio (entities, repositories, useCases)
- src/data: Implementación de datos (models, repositories, sources)
- src/presentation: Interfaz de usuario (components, navigation, screens, store)
- src/core: Utilidades y hooks globales

## Stack
- Expo (React Native)
- Supabase (backend)
- Zustand (estado global)
- Expo Router (navegación)
- NativeWind (estilos)
- Lottie React Native (animaciones)
- Leaflet + OpenStreetMap (mapas)
- Gemini API (asistente virtual)
